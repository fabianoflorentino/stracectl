Everything you need to
understand your syscalls

Built for developers and SREs who need signal, not noise.

📊

Real-time Aggregation

Syscalls counted, timed, and grouped as they happen. No log file, no post-processing — just live signal.

⏱

Latency Columns

AVG, MAX, TOTAL, P95, and P99 per syscall. MAX exposes outliers that averages always hide.

🚨

Smart Anomaly Detection

Rows turn red when ERR% ≥ 50 %, yellow when AVG ≥ 5 ms. Human-readable explanations at the bottom.

🌐

HTTP Sidecar Mode

--serve :8080 exposes JSON, WebSocket, and Prometheus endpoints. Live HTML dashboard included.

☸️

Kubernetes-Ready

Dockerfile, raw manifests, and a Helm chart with hardened sidecar security context. Discover PIDs across namespace boundaries.

📄

HTML Report Export

--report report.html writes a self-contained, sortable HTML file — no server, no CDN, no stracectl needed to view it.

Per-errno Breakdown

See exactly how many failures map to ENOENT, EACCES, EAGAIN — not just the aggregate error count.

Post-mortem Analysis

Three modes,
one tool

01

TUI Mode

Run or attach to any process. The htop-style dashboard updates live every 500 ms — sort, filter, explore details without leaving the terminal.

sudo stracectl run nginx
sudo stracectl attach 1234
02

Sidecar / HTTP API Mode

Add --serve :8080 to replace the TUI with an HTTP server — JSON at /api/syscalls, WebSocket live feed, Prometheus metrics, and a React-free live dashboard at /.

sudo stracectl attach --serve :8080 1234
03

Post-mortem / Stats Mode

Already have a strace -T -o log? Feed it to stracectl stats for the same TUI, HTTP API, or HTML report — no live process needed.

stracectl stats --report out.html trace.log

Up and running
in seconds

Linux only (uses ptrace via the strace binary). Requires Go 1.26+ or pull the pre-built container image.

Full install guide → 
go install github.com/fabianoflorentino/stracectl@latest
git clone https://github.com/fabianoflorentino/stracectl
cd stracectl
go build -o stracectl .
sudo mv stracectl /usr/local/bin/
docker pull ghcr.io/fabianoflorentino/stracectl:latest
